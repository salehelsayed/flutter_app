## Notes on generated assets (critical)

The file `assets/js/core_lib.js` is generated by bundling the TypeScript sources in `core_lib_js/`. After modifying any `.ts` file in `core_lib_js/src/`, you **must** rebuild:

```bash
cd core_lib_js
npm run build
```

This outputs the bundled `core_lib.js` to `assets/js/`. Without this step, changes to TypeScript code will not be available at runtime.
